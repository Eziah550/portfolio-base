## 📋 What is portfolio-base?

portfolio-base is a minimalist and reusable portfolio template designed for developers. It helps you create a clean online portfolio quickly. You can clone the template, customize it with your own details, and publish it online in just a few minutes. This tool is ideal if you want a basic, professional portfolio without the hassle of building it from scratch.

---

## 💻 Who is this for?

You do not need any coding skills to use portfolio-base. This guide will help you get your own portfolio website up and running even if you have little to no technical experience. If you have used websites to browse or shop online, you can follow these instructions.

- **Text editor (optional):** Programs such as Notepad (Windows), TextEdit (Mac), or any simple text editor to customize your portfolio.
- **Web browser:** Chrome, Firefox, Edge, Safari, or others to view your portfolio.

The portfolio-base package contains all the files needed for your website. These files are made in web languages (HTML, CSS, and JavaScript). When you open these files in a browser, they display your portfolio. You can change the text and images inside these files to make the portfolio your own.
